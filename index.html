② GitHub Pages で公開 恒久URL・無料

一度設定すれば https://あなた名.github.io/jpa-padel/ という永続URLができます。 スマホのQRコードで選手に配布できます。

設定手順（初回のみ・約5分）
1. github.com で無料アカウント作成
2. 「New repository」→ 名前を jpa-padel にして作成
3. 「Add file」→「Upload files」→ このHTMLファイルを index.html という名前でアップロード
4. Settings → Pages → Branch: main → Save
5. 数分後に https://ユーザー名.github.io/jpa-padel/ でアクセス可能
6. スコアを更新したら、このHTMLファイルを上書きアップロードするだけ
※ GitHubのファイルはすべて公開されます（大会データが見えます）

③ リアルタイム自動同期 サーバー環境が必要

JSONBin.io APIを使います。
⚠️ ファイルをダブルクリックで開く（file://）場合は動作しません。
GitHub Pages等のURL（https://）からアクセスした場合のみ使えます。

設定手順：
1. jsonbin.io で無料アカウント作成
2. API Keys → Secret Key をコピー
3. 下に貼り付けて「同期開始」
